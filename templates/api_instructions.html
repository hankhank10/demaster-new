API Instructions

Endpoint

http://demaster.hankapi.com/demaster

Accepted arguments

?long_track_name=

Will accept text including spaces, hyphens and brackets

?format=

html : serves human friendly output (default if not specified)
json : generates json (example here)
simple : returns the short title in plain text
